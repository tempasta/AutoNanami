<h1>AutoNanami</h1>

an external memory macro, written in python, for aba (anime battle arena), that automatically hits nanami's timings by reading memory instead of using pixel or image scanning

### features

* **dynamic tracking**  
  automatically scans and targets entities while ignoring other people's timings

* **live stats**  
  displays active target, real time scale, ratio counter, and black flash state

* **hit chance**  
  a percentage slider (0% to 100%) that randomizes click consistency  

* **global toggle**  
  a master toggle to stop the script

### quick tutorial
(make sure you have [python](https://www.python.org/ftp/python/3.14.5/python-3.14.5-amd64.exe) before proceeding!)
#### 1. download
download `source code (zip)` from the [latest release](https://github.com/tempasta/AutoNanami/releases) and extract

#### 2. install dependencies
run `requirements.bat` to install the required packages (you only have to do this once)

#### 3. open roblox
roblox HAS to be open before the script!

#### 4. run the script
run `main.py` to start the script
